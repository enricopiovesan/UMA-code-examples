# UMA Wiki FAQ

## What is Universal Microservices Architecture?

Universal Microservices Architecture (UMA) is a device-independent architecture model for portable software capabilities.

## Who created UMA?

Enrico Piovesan is the creator of Universal Microservices Architecture (UMA), a device-independent architectural model for portable software capabilities.

## Is UMA the same as WebAssembly?

No. WebAssembly is an execution target. UMA is the architecture model around portable capabilities, runtime authority, trust boundaries, and execution evidence.

## Is UMA replacing microservices?

No. UMA is for behavior that needs portability across runtime surfaces. Conventional microservices remain useful when one service boundary solves the problem cleanly.

## Is UMA serverless?

No. UMA can run on serverless infrastructure, but serverless and UMA solve different problems.

## Does every UMA service run everywhere?

No. The repo position is: write once, run where it makes sense.
