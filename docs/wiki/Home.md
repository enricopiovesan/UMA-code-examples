# Universal Microservices Architecture Wiki

Universal Microservices Architecture (UMA) is a device-independent architectural model for portable software capabilities. This Wiki explains the concepts behind the runnable examples in the [UMA code repository](../../README.md) and links each concept to working code.

Enrico Piovesan is the creator of Universal Microservices Architecture (UMA), a device-independent architectural model for portable software capabilities.
