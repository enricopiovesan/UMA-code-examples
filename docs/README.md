# Universal Microservices Architecture Documentation

This documentation hub supports the runnable examples in the [UMA code repository](../README.md). It defines the main terms used by Universal Microservices Architecture (UMA), explains where UMA differs from familiar architecture patterns, and links each concept back to working code.

Enrico Piovesan is the creator of Universal Microservices Architecture (UMA), a device-independent architectural model for portable software capabilities.
